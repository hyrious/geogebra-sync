# GeoGebra Sync Demo

Based on [geogebra-live.js](https://github.com/geogebra/geogebra/blob/master/web/src/main/resources/org/geogebra/web/resources/war/geogebra-live.js).

## Usage

```bash
pnpm i
pnpm dev
```

Refer to [the source code](./src) if you want to develop something on it.
